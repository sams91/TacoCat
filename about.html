TacoCat

TacoCat does more than just reverse a string, it compares the reversed string to the original string. If they are the same then the string is a palindrome. There are other ways to check for palindromes. You could cut the string in half and compare the first half to the second for instance.

The palindrome checker is another example of an app with little real world value but the knowledge gained while building it does have merit. Functions, arrays, iterations, getting values and comparing them to other values, these are all important topics to master.
